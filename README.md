# Deadline

Deadline turns a course syllabus PDF into calendar events. Upload your syllabus, review the extracted assignments and due dates, then download a `.ics` file or push everything directly to Google Calendar.

## Features

- **PDF upload** — drop in any syllabus PDF and get a structured list of assignments and deadlines back in seconds.
- **AI extraction** — Gemini reads the syllabus and returns structured data: assignment name, type, and due date.
- **Review before export** — see every extracted item before committing; items with unrecognized dates are flagged separately.
- **ICS download** — export as a standard `.ics` file that imports into any calendar app.
- **Google Calendar push** — connect your Google account and push all dated events directly to your calendar in one click.
- **No account needed** — there's no sign-up; Google OAuth is only used for the Calendar push and tokens are scoped to calendar events only.

## How it works

1. **Extract** — the PDF is parsed with pdfplumber and the raw text is sent to Gemini, which returns a JSON array of schedule items (name, type, due date).
2. **Review** — the frontend displays the items in a review screen. Undated items are shown separately since they can't be placed on a calendar.
3. **Export** — on download, a `.ics` file is generated by hand per RFC 5545: all-day events to avoid timezone bugs, with undated items written as `VTODO` entries. On Google push, the app authenticates via OAuth 2.0 (authorization code + PKCE) and creates all-day events via the Calendar API.

## Stack

- **Backend:** Python + Flask
- **Frontend:** HTML/CSS + Alpine.js
- **AI:** Google Gemini
- **PDF parsing:** pdfplumber
- **Calendar:** Google Calendar API (OAuth 2.0), RFC 5545 ICS
